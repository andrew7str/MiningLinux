# How To Use For Linux
* sudo apt update
* sudo apt install -y git build-essential cmake libuv1-dev libssl-dev libhwloc-dev
* chmod +x *
* ./start_mining.sh

# How To Use For Windows
* Download File For XMRING Windows
* run file "start_mining.bat"
